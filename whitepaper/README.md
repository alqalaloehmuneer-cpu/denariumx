# DenariumX Whitepaper

This directory contains the official whitepaper of **DenariumX**, a time-governed monetary protocol
in which temporal correctness is a prerequisite for monetary validity.

## Contents

- **whitepaper.md**  
  The canonical Markdown version of the whitepaper, suitable for GitHub reading and version control.

- **whitepaper.pdf**  
  The designed PDF version intended for publication, sharing, and formal review.

- **figures/**  
  Diagrams and figures referenced by the whitepaper, including:
  - Time-Governed Block Issuance & Validation
  - Temporal Manipulation Detection & Block Burn Mechanism

## Purpose

The whitepaper documents:
- The motivation behind DenariumX
- The time-governed issuance model
- Temporal enforcement and block burn logic
- Economic and security implications

This document is **non-commercial** and **non-speculative**.
It is intended for research, protocol design discussion, and long-term experimentation.
